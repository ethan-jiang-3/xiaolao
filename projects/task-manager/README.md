# Task Manager - 三虾协作实验项目

验证三虾协作 + GitHub Flow + TDD 的完整流程。

## 项目规则（重要）

**每个实验项目必须遵循：**
1. **一个 project 一个目录** - 所有代码、测试、文档都在该目录内
2. **模块化分工** - 不同 Agent 负责不同模块，但在同一目录下开发
3. **统一入口** - 一个 package.json，一个测试配置
4. **README 即文档** - 项目说明、规则、完成情况都写在这里

## 模块分工

| 模块 | 路径 | 负责人 | 功能 |
|------|------|--------|------|
| 核心引擎 | `src/engine/` | 虾老大 | Task、TaskManager、事件系统 |
| CLI 界面 | `src/cli/` | 虾可爱 | 命令行交互、格式化输出 |
| 数据存储 | `src/storage/` | 虾算盘 | JSON 存储、统计分析 |

